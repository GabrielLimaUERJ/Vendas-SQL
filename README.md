# Análise de Vendas (SQL + Streamlit)

Aplicação em Python que realiza análises de dados de vendas
utilizando consultas SQL em banco de dados MySQL e visualização interativa com Streamlit.

## Tecnologias

- Python
- Streamlit
- Pandas
- MySQL
- SQL

## Funcionalidades

- conexão com banco de dados MySQL
- consultas SQL dinâmicas
- análise de receita e lucro
- identificação de produtos e clientes com prejuízo
- ranking de clientes e produtos
- análise por região
- visualização interativa com gráficos

## Análises realizadas

- Receita por categoria
- Lucro por categoria
- Produtos com prejuízo
- Clientes com prejuízo
- Top clientes
- Produtos mais vendidos
- Vendas por região
- Ticket médio por cliente
- Relação entre desconto e lucro

## Objetivo

Simular um ambiente real de análise de dados,
integrando SQL com Python para geração de insights de negócio.
